# Frontend - Sistema de Presupuestos

Interfaz local para administrar recursos, APUs, presupuestos y control de costos.

## Arranque

```powershell
npm run dev
```

## Verificacion

```powershell
npm run lint
npm run build
```

## Guia UI

Las reglas visuales del frontend estan documentadas en:

- [UI_GUIDE.md](./UI_GUIDE.md)

Antes de crear o modificar pantallas, revisar esa guia y reutilizar los componentes de `src/components/ui.jsx`.
